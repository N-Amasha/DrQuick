# DrQuick – Smart Doctor Appointment Booking System

DrQuick is a web-based doctor appointment booking system built using PHP and MySQL.  

## Day 1 Progress
- Project folder structure created
- User registration (Patient / Doctor)
- Secure login system with password hashing
- Role-based redirection:
  - Doctor → /doctor/index.php
  - Patient → /patient/index.php
- Session-based authentication and login attempt limits

## Tech Stack
- PHP
- MySQL
- HTML / CSS
- XAMPP

## Next Steps
- Doctor schedule management
- Patient appointment booking
- Appointment approval/rejection
- Notifications
