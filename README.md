# DrQuick – Smart Doctor Appointment Booking System

DrQuick is a web-based doctor appointment booking system built using PHP and MySQL.  
This project focuses on secure authentication, role-based access control, and a clean backend architecture.

## Features (Current)
- User registration (Doctor / Patient)
- Secure login with password hashing
- Login attempt limiting
- Role-based redirection
- Session-based access control
- Modular folder structure

## Tech Stack
- PHP
- MySQL
- HTML, CSS
- Apache (XAMPP)

## Project Status
In Progress – Core authentication completed

## Next Planned Features

- Patient appointment booking
- Appointment approval & management
